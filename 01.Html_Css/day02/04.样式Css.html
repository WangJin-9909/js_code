样式的形式

  • 1.行内样式表或者行内样式
  • 2.内部样式表，写在当前文件中，一般再head标签内
  • 3.外部样式表，在head标签中声明即可，使用link标签，但标签

    • 行内样式

    font-size 文字大小 em、px:像素、pt:点、cm:厘米、in:英寸
    font-family 字体
    font-weigh 字体加粗 取值：normal[常用不加粗]、bold[常用加粗]、bolder、lighter、number
    font-style 字体风格：正常、斜体 取值normal正常字体、italic斜体
    line-height 行间距
    text-align 水平对齐方式 取值：left、center、right
    text-ident 首行缩进 值为2em 段落缩进2个字
    color 文字颜色
