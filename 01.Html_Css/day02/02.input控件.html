type取值

  • type="text" 默认文本
  • type="password" 密码框
  • type="radio" 单选框 同一组name需要一致
  • type="check" 复选框 同一组复选框name需要一致
  • type="button"
  • type="submit" 提交按钮，用于表单提交
  • type="reset" 复位按钮，用于表单复位
  • type="image" 图像形式的提交按钮
  • type="file" 上传文件

input其它属性

属性 取值 描述
name 用户自定义 控件名称
value 用户自定义 input空间中的默认文本值
size 正数，一般不使用，通过css来控制 input控件的显示宽度
checked checked 定义控件默认是否被选中
maxLength 正整数 控件允许输入的最多字符数
