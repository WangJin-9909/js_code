# window事件：

onload()：页面加载完成回调

1. ##### 点击事件：

    onclick       单击事件

    ondbclick   双击事件

2. ##### 焦点事件

   onblur     失去焦点

   onfocus  元素获得焦点

3. 加载事件

   onload    页面加载完成事件

4. 鼠标事件

   onmousemove     鼠标被移动    

   onmouseout         鼠标从某元素移开

   onmouseover       鼠标移到某元素之上

   onmousedown      鼠标按钮按下

   onmouseup           鼠标按键松开

5. 键盘事件

   onkeydown            键盘按下

   onkeyup                 键盘抬起

   onkeypress            键盘按下并抬起

6. 选中和改变

   onchange            域的内容被改变

   onselect              文本被选中

7. 提交

   onsubmit             提交事件
